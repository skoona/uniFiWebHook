* ILI9488 16-bit parallel
* FT6236 Touch
* esp32s3
* Espressif v5.5.2
* Lvgl v9.4

## Function
Implements Webhook listener from UniFi Protect and displays the snapshot of the alert on the display.

Uses the UniFi Protect API to collect camera snapshots for two specific cameras, based on screen touches.
